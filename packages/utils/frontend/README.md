# @o2s/utils.frontend

Utility functions for O2S frontend applications, including JWT and date helpers.

## Installation

```bash
npm install @o2s/utils.frontend
```

## Peer Dependencies

- `jsonwebtoken` ^9.0.2
- `dayjs` ^1.11.13
